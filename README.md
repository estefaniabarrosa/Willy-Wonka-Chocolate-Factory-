#  Willy Wonka Chocolate Factory – SQL From Data to Insight

##  Project Overview

This project was developed as part of the **Ironhack Data Analytics Bootcamp** mini project:  
**SQL: From Data to Insight**.

The goal of this project is to transform a raw transactional dataset into a structured relational database and use **SQL + Python** to generate actionable business insights for the fictional **Willy Wonka Chocolate Factory**.

Our analysis focuses on identifying:

- High-performing products
- Regional sales opportunities
- Factory profitability
- Product performance patterns
- Business optimization opportunities

---

# Business Problem

The Willy Wonka Chocolate Factory wants to improve profitability and operational efficiency by understanding:

- Which products generate the highest revenue and profit
- Which regions contribute most to sales performance
- Whether some factories underperform compared to others
- Which products show lower profitability despite generating sales

The company aims to use data-driven decision-making to optimize:

- Product strategy
- Inventory allocation
- Regional focus
- Factory performance

---

#  Research Questions

## RQ1
Which chocolate products generate the highest revenue and profit?

## RQ2
Which regions contribute most to overall sales performance?

## RQ3
Are some factories significantly less profitable than others?

## RQ4
Which products show weaker profitability margins despite generating sales?

---

# Hypotheses

## H1
Premium chocolate bar products generate significantly higher profit margins than smaller candy products.

## H2
Some regions consistently outperform others in overall sales and profitability.

## H3
Certain factories contribute disproportionately to lower profitability levels.

#  Dataset Information

## Main Dataset
- Willy Wonka Chocolate Factory transactional dataset (`.csv`)

## Dataset Characteristics

The dataset contains:

- Transactional sales records
- Customer information
- Product information
- Geographic data
- Sales metrics
- Profitability metrics
- Factory information
- Shipping information

Each row in the original dataset represents a sales transaction.
